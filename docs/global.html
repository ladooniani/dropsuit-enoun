enoun_f(input, swa, dispout) → {object}

Separates English nouns and stop words from an input string.
Parameters:
Name Type Description
input string The input sentence raw string. Pass NULL to process the constructor's 'input'.
swa number An integer in the range of 0-2, which determines the amount of stop words to be processed. 0 processes the minimum amount, 1 adds extra stop words, and 2 includes the maximum number of stop words from three distinct lists.
dispout boolean Determines if the processing output results are displayed in the terminal.

Returns:
- Returns an object with the following options: - noun(): Separated English nouns - stop(): Rest of stop words
